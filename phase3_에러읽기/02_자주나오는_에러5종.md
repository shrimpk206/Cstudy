## 1) 🔧 TypeError — "이거 이렇게 쓰면 안 되는데..."

```
TypeError: Cannot read properties of undefined (reading 'name')
TypeError: someFunction is not a function
TypeError: Cannot read properties of null (reading 'map')
```

### 무슨 뜻?

값에 대해 **그 타입에 맞지 않는 동작**을 했어요.

### 자주 일어나는 상황

- 객체가 `undefined`인데 속성 꺼내려 함 → `user.name` (user가 undefined)
- 함수가 아닌데 호출 → `data.filter()` (data가 함수 없는 객체)
- 배열인 줄 알았는데 객체 → `users.map()` (users가 객체)

### 고치는 법

```javascript
// ❌
const name = user.name

// ✅ 옵셔널 체이닝
const name = user?.name

// ✅ 또는 명시적 검사
const name = user ? user.name : "이름없음"
```

### AI한테:

> "TypeError: Cannot read properties of undefined (reading 'X'). users.js:12에서. 그 줄: `user.X`. user가 undefined일 가능성? 안전 처리 추가해줘."

---

## 2) 👻 ReferenceError — "그런 거 본 적 없는데?"

```
ReferenceError: foo is not defined
```

### 무슨 뜻?

그 이름의 변수/함수를 **찾을 수 없어요.**

### 자주 일어나는 상황

- 🅰️ **오타**: `consle.log` (console 아니고 consle)
- 📥 **import 누락**: 다른 파일 함수를 import 안 함
- 🔮 **스코프 문제**: `if {}` 안에서 만든 변수를 밖에서 씀

### 고치는 법

이름 오타 확인 → import 점검 → 변수 선언 위치 점검!

### AI한테:

> "ReferenceError: foo is not defined — main.js:5에서. foo는 utils.js에 있는데 import 빠진 듯. 추가해줘."

---

## 3) 🔒 SyntaxError — "이게 무슨 말이야?"

```
SyntaxError: Unexpected token '}'
SyntaxError: Unexpected end of input
SyntaxError: Invalid or unexpected token
```

### 무슨 뜻?

코드를 읽다가 **문법이 깨진 곳** 발견. **실행도 못 들어가요.**

### 자주 일어나는 상황

- 괄호/중괄호/대괄호 짝 안 맞음 (`{`, `}` 짝)
- 따옴표 안 닫힘
- 쉼표 위치 이상
- 영어 코드인데 한글 따옴표(`"`)가 섞임 (이거 진짜 자주 함!)

### 고치는 법

- 에디터의 자동 들여쓰기 체크 (어긋나면 짝 안 맞은 거)
- ESLint/Prettier 활용
- "Unexpected end of input"이면 끝까지 안 닫힘

### AI한테:

> "SyntaxError. 어디가 문제인지 모르겠어. 전체 코드 붙여줄게: [코드]. 문법 오류 찾아줘."

---

## 4) 📦 Module Not Found — "그 친구 어디 갔지?"

```
Error: Cannot find module 'axios'
Module not found: Can't resolve './utils'
ModuleNotFoundError: No module named 'requests'   ← Python
```

### 무슨 뜻?

import한 모듈을 **못 찾아요.**

### 자주 일어나는 상황

- 라이브러리 **설치 안 함** → `npm install axios` 빠짐
- **경로 오타** → `./utlis` (utils 아니고 utlis)
- **대소문자** 틀림 → `./Button`인데 파일은 `button.js` (Mac/Linux는 구분!)
- **확장자** 빠짐 → 환경에 따라 `./utils.js`로 명시 필요

### 고치는 법

- 외부 라이브러리: `npm install xxx` 또는 `pip install xxx`
- 본인 파일: 경로 / 대소문자 / 확장자 점검

## 5) 📡 Network / API 에러 — "전화 안 받아!"

```
Failed to fetch
NetworkError when attempting to fetch resource
Request failed with status code 401
CORS policy: No 'Access-Control-Allow-Origin' header
```

### 자주 일어나는 상황

| 메시지 | 의미 |
|---|---|
| **CORS 에러** | 브라우저에서 다른 도메인 호출 → 보안 정책으로 막힘 |
| **401 Unauthorized** | API 키/토큰 빠짐 |
| **404** | URL 오타 |
| **Failed to fetch** | 인터넷 끊김 / 서버 다운 / URL 없음 |

### ⚠️ CORS 에러는 특별 안내

이건 **백엔드 문제거나, 프록시를 둬야 해결됩니다**. 프론트엔드 코드만 고쳐서는 안 풀려요!

- 본인 백엔드 → CORS 헤더 추가
- 남의 API → 본인 백엔드(서버) 거쳐서 호출

### AI한테:

> "API 호출 CORS 에러. 브라우저에서 https://other-api.com 직접 호출 중. 우리 백엔드(Express)에서 프록시 엔드포인트 만들어 거기로 보내게 바꿔줘."

## 🩺 빠른 진단표 — 응급실에 비유

| 첫 단어 | 1순위 의심 | 처방 |
|---|---|---|
| 🔧 TypeError | 값이 undefined/null이거나 타입 불일치 | 옵셔널 체이닝 (`?.`), 타입 검사 |
| 👻 ReferenceError | 오타, import 누락, 스코프 | 이름 확인, import 추가 |
| 🔒 SyntaxError | 괄호/따옴표 짝 | 들여쓰기로 추적 |
| 📦 Cannot find module | 설치 안 했거나 경로 틀림 | install / 경로 수정 |
| 📡 Failed to fetch / CORS | 네트워크/인증/도메인 | 환경변수, 프록시, 백엔드 |

## 🎯 확인해봐요

각 에러의 종류와 가장 의심해야 할 원인을 한 줄로!

1. `TypeError: result.map is not a function`
2. `ReferenceError: useStat is not defined`
3. `Module not found: Error: Can't resolve 'axios'`

<details>
<summary>👀 정답 보기</summary>

1. **TypeError**. result가 배열이 아닌데 `.map()`을 호출. 응답 형식 점검 (혹시 객체로 옴? 비어 있음?).
2. **ReferenceError**. 거의 확실히 **`useState` 오타**!
3. **Module not found**. **`npm install axios`** 안 했음.

</details>
