# 02. 동작을 묶는 단위 🥤

## 한 줄로 말하면

**함수 = 자판기예요.**
동전 넣으면 → 음료 나옴. 똑같아요.

---

## 🥤 자판기를 떠올려봐요

자판기 앞에서 일어나는 일:

1. 동전을 **넣어요** (입력)
2. 자판기가 **뭔가 해요** (처리)
3. 음료가 **나와요** (결과)

함수가 정확히 이거예요.

```javascript
function add(a, b) {        // a, b가 동전
  return a + b              // 결과가 음료
}

const result = add(3, 5)    // 음료 = 8
```

| 자판기 | 함수 |
|---|---|
| 동전 | **인자** (parameter) |
| 음료 | **반환값** (return) |
| 자판기 작동 | **함수 호출** (call) |

---

## 1) 함수 만들기

```javascript
function double(x) {
  return x * 2
}

double(5)    // 10
double(7)    // 14
double(100)  // 200
```

`double`이라는 자판기 하나를 만들어두면, 동전 바꿔가며 계속 쓸 수 있어요.

> 💡 **재사용**이 핵심! 같은 동작을 여러 번 쓸 일이 있으면 자판기로 만들어두세요.

---

## 2) 입력이 없거나, 결과가 없거나

### 동전 안 넣어도 되는 자판기 (무료 음료기)

```javascript
function sayHello() {              // 입력 없음
  console.log("안녕!")             // 결과도 없음, 그냥 외쳐요
}

sayHello()    // 안녕!
```

### 음료가 안 나오는 자판기 (그냥 놀이용)

```javascript
function celebrate() {
  console.log("🎉 잘했어요!")
  // return이 없음
}
```

---

## 3) 화살표 함수 — 짧게 쓰는 법 ⚡

```javascript
const double = (x) => x * 2
const add = (a, b) => a + b
const greet = name => "안녕, " + name
```

같은 함수예요. **그냥 모양만 짧게**.

JavaScript에서 정말 자주 봐요. 이렇게 생긴 거 보면 "아, 함수구나" 하면 됩니다.

---

## 4) 함수가 함수를 부를 수도 있어요

```javascript
function double(x) { return x * 2 }
function quadruple(x) { return double(double(x)) }

quadruple(3)   // 12
//   ↓
//   double(double(3))
//   double(6)
//   12
```

자판기 안에 또 자판기가 있는 셈! **큰 일을 작게 쪼개기**의 기초예요.

## 🤖 AI한테 부탁할 때

| 이런 상황 | 이렇게 말해보세요 |
|---|---|
| 한 함수가 200줄 넘게 큼 | "이 함수를 작은 함수 3-4개로 쪼개줘. 각 함수는 한 가지 일만" |
| 비슷한 코드가 여러 군데 반복 | "이 부분을 함수로 빼서 재사용하게 해줘" |
| 함수명이 모호함 | "함수명을 동사로, 무슨 일 하는지 드러나게 (`process` → `convertCsvToJson`)" |
| AI 함수가 이상하게 동작 | "이 함수에 `add(3, 5)`를 넣으면 뭐가 나와야 해?"라고 검증 |

### 🗣️ 정확한 단어 쓰기

AI한테 말할 때 단어를 정확히 쓰면 의사소통이 잘 돼요:

- "함수를 **호출한다**" = 자판기 작동시킨다
- "함수를 **정의한다**" = 자판기를 만든다
- "함수가 **반환한다**" = 자판기에서 음료가 나온다

## 🎯 확인해봐요

```javascript
function greet(name, lang) {
  if (lang === "ko") return "안녕, " + name
  return "Hello, " + name
}
```

1. 인자, 반환값, 함수 이름을 짚어보세요.
2. `greet("철수", "ko")`의 결과는?

2. **`"안녕, 철수"`**
