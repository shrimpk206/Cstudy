# 03. 흐름 제어 🚦

## 한 줄로 말하면

코드는 위에서 아래로 한 줄씩 읽혀요.
그 흐름을 **갈라지게** 하거나 **반복하게** 만드는 거예요.

---

## 1) if문 — 신호등이에요 🚦

```javascript
const age = 20

if (age >= 19) {
  console.log("성인")
} else if (age >= 13) {
  console.log("청소년")
} else {
  console.log("어린이")
}
```

```
🚦 if(조건)     → 초록불 (조건 맞음? 안으로!)
🚦 else if      → 노란불 (위가 아니면 이건?)
🚦 else         → 빨간불 (다 아니면)
```

위에서 아래로 차례차례 검사해요. **하나가 맞으면 거기서 끝.**

---

## 비교 연산자 — 신호 모양

| 기호 | 뜻 | 예 |
|---|---|---|
| `===` | 같다 | `age === 20` |
| `!==` | 다르다 | `age !== 20` |
| `>` | 크다 | `age > 19` |
| `<` | 작다 | `age < 13` |
| `>=` | 크거나 같다 | `age >= 19` |
| `<=` | 작거나 같다 | `age <= 13` |

> ⚠️ JavaScript에서 `==` 말고 **`===`** 쓰세요! `==`는 함정 많아요.

---

## 그리고 / 또는

```javascript
if (age >= 19 && hasId) { ... }      // && = 그리고
if (role === "admin" || isOwner) { } // || = 또는
if (!isLogged) { ... }                // ! = 부정
```

| 기호 | 뜻 |
|---|---|
| `&&` | **그리고** (둘 다 맞아야) |
| `\|\|` | **또는** (하나만 맞아도) |
| `!` | **아니다** |

자판기 살 때:
- "동전이 있고 **&&** 음료가 있으면" → 살 수 있어요
- "콜라 **\|\|** 사이다면" → 둘 중 아무거나 OK

---

## 2) for문 — 빨래 개기 🧺

빨래 개야 할 옷이 5개예요. 같은 동작 5번. 이게 반복문이에요.

```javascript
for (let i = 0; i < 5; i++) {
  console.log(i)
}
// 0, 1, 2, 3, 4 출력
```

해석:
- `let i = 0`: 0번부터 시작
- `i < 5`: 5보다 작을 동안
- `i++`: 한 번 끝날 때마다 1 증가

처음엔 외계어 같지만 99% 이 모양이에요.

---

## for...of — 더 쉬운 반복

배열 하나씩 꺼낼 때:

```javascript
const fruits = ["사과", "바나나", "포도"]

for (const fruit of fruits) {
  console.log(fruit)
}
// 사과
// 바나나
// 포도
```

훨씬 직관적이죠? 이게 더 자주 쓰여요.

## 3) 배열 메서드 — 진짜 자주 보는 마법 ✨

이 4개는 외워두세요. 진짜진짜 자주 나와요.

### map — **변환**

```javascript
const numbers = [1, 2, 3, 4]
const doubled = numbers.map(n => n * 2)
// [2, 4, 6, 8]
```

각 항목을 **변환**해서 새 배열로!

### filter — **거르기**

```javascript
const evens = numbers.filter(n => n % 2 === 0)
// [2, 4]
```

조건에 **맞는 것만** 남겨요.

### reduce — **합치기**

```javascript
const sum = numbers.reduce((a, b) => a + b, 0)
// 10
```

다 합쳐서 **하나의 값**으로.

### forEach — **그냥 한번씩**

```javascript
numbers.forEach(n => console.log(n))
// 1, 2, 3, 4 출력 (반환값 없음)
```

그냥 **각자 뭔가 시키기**.

> 💡 **map = 변환, filter = 거르기, reduce = 합치기, forEach = 한번씩.** 이거만 외워도 절반은 끝!

| 무한 루프(영원히 안 끝남) | "이 while문 언제 끝나? 어떤 변수가 어떻게 바뀌어야 종료?" |

## 🎯 확인해봐요

```javascript
const items = [10, 20, 30]
const result = items.filter(n => n >= 20).map(n => n + 1)
console.log(result)
```

1. `result`는 뭐가 나올까요?
2. `&&`와 `||` 차이는?

<details>
<summary>👀 정답 보기</summary>

1. **`[21, 31]`** — 20 이상만 남기고(20, 30) 각각 1을 더해서.
2. `&&`는 양쪽 다 참이어야 참, `||`는 하나라도 참이면 참.

</details>
