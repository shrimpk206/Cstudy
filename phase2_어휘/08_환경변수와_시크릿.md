# 08. 환경변수와 시크릿 🔐

## 한 줄로 말하면

API 키나 비밀번호를 코드에 박으면 **깃허브 올라가는 순간 털려요.**
**비밀 일기장**에 따로 적어두세요.

---

## 🔐 비밀 일기장 비유

학교에서 친구한테 일기장을 보여주면? 큰일이죠.
코드도 똑같아요. 누구나 보는 깃허브에 비밀번호가 올라가면 **하루 안에 털려요.**

진짜로 일어나는 일이에요. 봇이 24시간 깃허브 새 커밋을 감시하고 있어요.

---

## 1) 무엇이 시크릿이죠?

이런 건 **절대로** 코드 파일에 직접 적지 마세요:

- 🔑 API 키 (OpenAI, Stripe, AWS 등)
- 🔒 데이터베이스 비밀번호
- 🎫 인증 토큰
- 💳 결제 시크릿

```javascript
// ❌ 절대 이렇게 하면 안 돼요
const apiKey = "sk-proj-abc123Real..."

// ✅ 환경변수에서 읽기
const apiKey = process.env.OPENAI_API_KEY
```

---

## 2) `.env` 파일 — 비밀 일기장 만들기 📔

프로젝트 루트에 `.env` 파일을 만들어요:

```
OPENAI_API_KEY=sk-proj-abc123Real
DATABASE_URL=postgres://user:pass@localhost/db
PORT=3000
```

규칙:
- `KEY=value` 형식
- 따옴표 안 써도 돼요
- 한 줄에 하나
- `=` 양옆 공백 없음

---

## 3) `.gitignore`에 추가 — 가장 중요!! ⚠️

```
# .gitignore
node_modules/
.env
.env.local
```

`.gitignore`는 **깃에 올리지 않을 파일 목록**이에요.
`.env`는 **반드시** 여기 있어야 해요.

> 🚨 안 적으면 깃허브에 키가 그대로 올라가요. **꼭 확인!**

---

## 4) 코드에서 사용하기

### JavaScript:

```javascript
// dotenv 라이브러리 (Node.js)
import "dotenv/config"

console.log(process.env.OPENAI_API_KEY)
```

Vite/Next.js는 `.env`를 자동으로 읽어요. 단, 프론트엔드에 노출용은 `VITE_` 또는 `NEXT_PUBLIC_` 접두사 필요.

### Python:

```python
import os
from dotenv import load_dotenv

load_dotenv()
api_key = os.getenv("OPENAI_API_KEY")
```

---

## 5) `.env.example` — 친구한테 빈 양식 주기

다른 사람이 프로젝트 받을 때 어떤 키가 필요한지 알려주는 파일:

```
# .env.example  (이건 깃에 올림)
OPENAI_API_KEY=
DATABASE_URL=
PORT=3000
```

값은 빈 채로, **이름만** 적어둬요.

---

## 6) 😱 이미 깃허브에 올라갔다면?

차분히, 빠르게:

1. **즉시 그 키 폐기(rotate)** ← 1순위! 서비스 대시보드에서 키 새로 발급받고 옛 키 무효화
2. git history에서 지운다고 해도 누가 봤을 수 있어요. **폐기가 우선**
3. `.gitignore`에 `.env` 추가하고 다시는 올라가지 않게

> 🔥 진짜 일어나는 일: 키 노출되고 1시간 안에 누군가 본인 OpenAI 계정으로 100만 원어치 GPT를 돌리고 있을 수 있어요. **폐기는 분 단위로 빨리.**

## 🤖 AI한테 부탁할 때

| 이런 상황 | 이렇게 말해보세요 |
|---|---|
| AI가 코드에 API 키 박았음 | "이 키를 환경변수로 빼줘. `.env`와 `.gitignore`도 만들어줘" |
| 새 API 연동했음 | "이 키는 `.env`에. `.env.example`에도 빈 값으로 추가" |
| 클라이언트(브라우저)에서 키 쓰려고 함 | "잠깐, 브라우저에 키 노출되면 위험한 거 아냐? 서버에서만 쓰게 해줘" ⚠️ |

### 💡 가장 자주 하는 실수

AI는 종종 **React/Vue 같은 프론트엔드에서 시크릿 키를 직접** 쓰려고 해요.

🚨 프론트엔드 코드는 사용자 브라우저로 다 전송됩니다 = **사용자가 다 볼 수 있어요!**

진짜 시크릿이 필요하면 **백엔드(서버)**를 거쳐서 호출해야 해요. AI에게:

> "프록시 엔드포인트 만들어줘. 백엔드에서 OpenAI 호출하고 결과만 프론트에 돌려줘"
